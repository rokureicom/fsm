# Installing

## Package Manager
You can add by git-url to Package Manager:
```
https://github.com/rokureicom/fsm.git
```

## Manifest
Or add to `Packages/manifest.json`:
```
"com.rokurei.fsm": "https://github.com/rokureicom/fsm.git",
```
